# Product

## Register

brand

## Users

ผู้บริหาร นักการตลาด และผู้อ่านข่าวธุรกิจไทยที่ต้องการข้อมูลสรุปเร็ว อ่านบน desktop เป็นหลัก (mockup สำหรับนำเสนอลูกค้าก่อนพัฒนา WordPress จริง)

## Product Purpose

Market Times เป็น mockup เว็บข่าวธุรกิจและการตลาด ใช้แสดงโครงสร้างหน้า Home, หมวดข่าว, บทความ และ Sidebar ให้ลูกค้าเห็นภาพก่อน go-live ความสำเร็จคือลูกค้าเห็น layout ชัด อ่านง่าย และรู้สึกเป็น media ระดับมืออาชีพ

## Brand Personality

น่าเชื่อถือ · ทันสมัย · เป็น editor — โทน magazine ธุรกิจไทย ไม่เล่นสีจัด ไม่ดูเป็น template สำเร็จรูป

## Anti-references

- เว็บข่าวที่ layout แน่นเกินไป ตัวอักษรเล็ก อ่านยาก
- AI landing page สไตล์ gradient ม่วง-ฟ้า glow
- Admin dashboard ที่ใช้ card ซ้อน card
- โฆษณาแทรกในเนื้อหาหลัก (โฆษณาเฉพาะ Sidebar)

## Design Principles

1. **Hierarchy first** — หัวข้อเด่นชัด รองรับการสแกนด่วน
2. **Editorial rhythm** — ช่องว่างและน้ำหนักตัวอักษรสลับกัน ไม่ monotonous
3. **Brand teal as accent** — สีเขียวจากโลโก้เป็น accent ไม่ใช่พื้นที่ทั้งหน้า
4. **Motion with purpose** — animation บอกลำดับและ feedback ไม่ใช่ตกแต่งเปล่า
5. **Thai readability** — line-height และขนาดตัวอักษรเหมาะกับภาษาไทย

## Accessibility & Inclusion

- WCAG AA สำหรับ contrast ข้อความหลัก
- รองรับ `prefers-reduced-motion` — ปิด entrance animation เมื่อผู้ใช้ตั้งค่า
- โฟกัส keyboard ชัดบนลิงก์และปุ่ม
